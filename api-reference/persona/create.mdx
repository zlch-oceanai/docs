## Endpoint

```http
POST /api/v1/music/persona
```

Creates a reusable AI singer persona from an existing clip. Once created, pass the returned `persona_id` to [Generate Music](/api-reference/music/generate) to apply the vocal style to any new track.

## Request body

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `name` | string | **Yes** | Display name for the persona (e.g. `"My Artist"` or `"цИСчЪДцнМцЙЛ"`). |
| `clip_id` | string | **Yes** | Source clip ID whose vocal style will be learned. |

## Response

```json
{
  "code": 0,
  "message": "ok",
  "request_id": "req-1710000000000",
  "data": {
    "taskBatchId": "batch-xyz789",
    "items": [
      {
        "id": "persona-abc123",
        "status": "complete"
      }
    ]
  }
}
```

| Field | Type | Description |
|-------|------|-------------|
| `taskBatchId` | string | Internal batch identifier |
| `items` | array | Created persona items |
| `items[].id` | string | **Persona ID** тАФ pass this as `persona_id` in generation requests |
| `items[].status` | string | `"complete"` when persona is ready to use |

## Using your Persona to generate music

Once you have a `persona_id`, pass it to any [Generate Music](/api-reference/music/generate) request. The AI will sing the generated track in the learned vocal style.

```bash
curl -X POST https://api.example.com/api/v1/music/generate \
  -H "Authorization: Bearer sk-mm-your-key" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "[Verse]\nцЬИхЕЙчЕзхЬичкЧхП░\nщгОш╜╗ш╜╗хР╣цЭе\n[Chorus]\nф╜ацШпцИСх┐Гф╕нчЪДхЕЙ",
    "title": "х┐Гф╕нчЪДхЕЙ",
    "tags": "ц╡БшбМ цКТцГЕ хе│хг░",
    "persona_id": "persona-abc123"
  }'
```

<Tip>
  `persona_id` works with both **custom mode** (`prompt` + `tags`) and **inspiration mode** (`gpt_description_prompt`). The vocal style is applied regardless of which generation mode you choose.
</Tip>

## Complete workflow

```
1. Record or generate a source clip with the target vocal style
        тЖУ
2. (Recommended) Run Stem Separation to isolate clean vocals
        тЖУ
3. POST /api/v1/music/persona  тЖТ  receive persona_id
        тЖУ
4. POST /api/v1/music/generate  +  persona_id  тЖТ  task_id
        тЖУ
5. Poll GET /api/v1/task/{task_id}  тЖТ  audio_url
```

## Create persona example

```bash
curl -X POST https://api.example.com/api/v1/music/persona \
  -H "Authorization: Bearer sk-mm-your-key" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Custom Singer",
    "clip_id": "abc123def456"
  }'
```

## Choosing a source clip

For best results, use a clip with:
- Clear, isolated vocals (minimal background noise)
- At least 30 seconds of vocal content
- A consistent vocal style throughout

<Tip>
  Run [Stem Separation](/api-reference/audio/stem) on a clip first to get a clean vocal track, then use that vocal audio as the source for persona creation.
</Tip>
